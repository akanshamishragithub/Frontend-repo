git branch - to check the branches available in local
create a new branch::  
git checkout -b branch_name
add the  file changes in the local::
git status
git add .
git add file_name
git status
commit the changes in the local::
git commit -m "updated file message"
git log
push the  code changes  into the github server::
git push origin branch_name
git log
